# VipVideo
系统里的接口失效了需要自己找<br>
系统之所以这么大，是因为加入了前端框架Zui
<br>嫌源码大的可以改成远程调用
<br>本系统花费3天时间加1个通宵写出来的
<br>你需要在DatabaseCon类里更改mysql密码，还有导入数据表,导入表之前请自己创立一个名字为video数据库 
<br>后台账号密码封装到了Admin类里。
<br>系统存在越权删除用户的BUG,留给需要学习的人去修复咯~
